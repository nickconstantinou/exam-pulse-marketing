# Personas Review Skill

## Purpose
Review marketing assets through the lens of 12 distinct EdTech personas to identify friction points, resonance issues, and commercial viability.

## How to Run a Review

### Step 1: Prepare the Asset
- Summarize the landing page/copy/feature in 200 words
- Include the CTA and key messaging
- Note target audience (age, exam board context)

### Step 2: Run Through Each Persona
For each persona, answer:
- **Reaction:** 1-sentence gut check in their voice
- **Verdict:** Pass / Fail / Friction
- **Feedback:** One actionable critique

### Step 3: Synthesize
Create an Executive Summary:
- **Conflict Map:** Where do personas disagree?
- **Vulnerability Report:** Accessibility, Integrity, Technical risks
- **Actionable Roadmap:** Quick Win, Retention Hook, Cheque-Writer Feature

## Review Triggers
Use this skill when:
- Launching new landing pages
- Testing ad copy
- Reviewing email sequences
- Evaluating new features
